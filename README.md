# 🧠 Neural Networks with PyTorch

A collection of popular Convolutional Neural Network (CNN) architectures implemented from scratch using the **PyTorch** framework.

The trained model weights for each architecture are available on Hugging Face.

---

## 🚀 Implemented Architectures

| Model                        | Notebook          | Trained Model                                                                                                |
| ---------------------------- | ----------------- | ------------------------------------------------------------------------------------------------------------ |
| LeNet-5                      | [`lenet.ipynb`](./lenet.ipynb)     | [https://huggingface.co/aijadugar/cifar-10-lenet5](https://huggingface.co/aijadugar/cifar-10-lenet5)             |
| AlexNet                      | [`alexnet.ipynb`](./alexnet.ipynb)   | [https://huggingface.co/aijadugar/cifar-10-alexnet](https://huggingface.co/aijadugar/cifar-10-alexnet)         |
| VGG16                        | [`vgg16.ipynb`](./vgg16.ipynb)     | [https://huggingface.co/aijadugar/cifar-10-vgg16](https://huggingface.co/aijadugar/cifar-10-vgg16)             |
| ResNet18                     | [`resnet18.ipynb`](./resnet18.ipynb)  | [https://huggingface.co/aijadugar/cifar-10-resnet18](https://huggingface.co/aijadugar/cifar-10-resnet18)       |
| DenseNet                     | [`densenet.ipynb`](./densenet.ipynb)  | [https://huggingface.co/aijadugar/cifar-10-densenet](https://huggingface.co/aijadugar/cifar-10-densenet)       |
| MobileNetV2                  | [`mobilenet.ipynb`](./mobilenet.ipynb) | [https://huggingface.co/aijadugar/cifar-10-mobilenet](https://huggingface.co/aijadugar/cifar-10-mobilenet) |
