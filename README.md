# 🧠 Neural Networks with PyTorch

A collection of popular Convolutional Neural Network (CNN) architectures implemented from scratch using the **PyTorch** framework.

The trained model weights for each architecture are available on Hugging Face.

---

## 🚀 Implemented Architectures

| Model                        | Notebook          | Trained Model                                                                                                |
| ---------------------------- | ----------------- | ------------------------------------------------------------------------------------------------------------ |
| LeNet-5                      | [`lenet.ipynb`](./lenet.ipynb)     | [https://huggingface.co/aijadugar/lenet-cifar10](https://huggingface.co/aijadugar/lenet-cifar10)             |
| AlexNet                      | [`alexnet.ipynb`](./alexnet.ipynb)   | [https://huggingface.co/aijadugar/alexnet-cifar10](https://huggingface.co/aijadugar/alexnet-cifar10)         |
| VGG16                        | [`vgg16.ipynb`](./vgg16.ipynb)     | [https://huggingface.co/aijadugar/vgg16-cifar10](https://huggingface.co/aijadugar/vgg16-cifar10)             |
| ResNet18 (Transfer Learning) | [`resnet18.ipynb`](./resnet18.ipynb)  | [https://huggingface.co/aijadugar/resnet18-cifar10](https://huggingface.co/aijadugar/resnet18-cifar10)       |
| DenseNet                     | [`densenet.ipynb`](./densenet.ipynb)  | [https://huggingface.co/aijadugar/densenet-cifar10](https://huggingface.co/aijadugar/densenet-cifar10)       |
| MobileNetV2                  | [`mobilenet.ipynb`](./mobilenet.ipynb) | [https://huggingface.co/aijadugar/mobilenetv2-cifar10](https://huggingface.co/aijadugar/mobilenetv2-cifar10) |
